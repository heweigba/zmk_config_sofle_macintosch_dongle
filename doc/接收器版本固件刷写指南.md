# 接收器版本固件刷写指南

## 架构说明

**接收器版本（dongle）由3个独立设备组成：**

1. **接收器（Dongle）** - 中心设备
   - 连接电脑（USB/蓝牙）
   - 接收左右手键盘的信号
   - 固件名称：`sofle_dongle`

2. **左手键盘** - 外设
   - 包含轨迹球
   - 通过蓝牙连接接收器
   - 固件名称：`sofle_dongle_left`

3. **右手键盘** - 外设
   - **包含小红点（Trackpoint）** ⚠️
   - 通过蓝牙连接接收器
   - 固件名称：`sofle_dongle_right`

## 小红点不能用的原因

### 问题分析

何威反馈：**"不刷接收器，小红点能用，刷完之后就不能用了"**

**原因**：
- ❌ **只刷了接收器**（sofle_dongle）
- ✅ **右手键盘还运行旧固件**
- ✅ **旧固件可能使用不同的通信协议或配置**

### 解决方案

**需要刷右手键盘固件！**

## 刷写步骤

### 1. 下载固件

从GitHub Actions下载firmware.zip，解压后包含以下文件：

```
firmware/
├── sofle_dongle/
│   └── sofle_dongle-u2f.bin          # 接收器固件
├── sofle_dongle_left/
│   ├── left_bbtrackball_dongle-u2f.bin  # 左手键盘（轨迹球）
│   └── settings_reset-u2f.bin           # 重置固件
├── sofle_dongle_right/
│   ├── right_trackpoint_dongle-u2f.bin  # 右手键盘（小红点）⚠️
│   └── settings_reset-u2f.bin           # 重置固件
```

### 2. 刷写顺序

**推荐顺序**：

#### 第1步：刷接收器（可选，如果之前已刷）
1. 将接收器进入bootloader模式
2. 刷入：`sofle_dongle-u2f.bin`

#### 第2步：刷右手键盘 ⚠️ **关键！**
1. 将右手键盘进入bootloader模式
2. **可选**：先刷`settings_reset-u2f.bin`清除旧配置
3. 刷入：`right_trackpoint_dongle-u2f.bin`
4. 重启右手键盘

#### 第3步：刷左手键盘（可选）
1. 将左手键盘进入bootloader模式
2. **可选**：先刷`settings_reset-u2f.bin`清除旧配置
3. 刷入：`left_bbtrackball_dongle-u2f.bin`
4. 重启左手键盘

### 3. 配对

1. 接收器连接电脑（USB或蓝牙）
2. 左右手键盘自动连接接收器
3. 测试小红点功能

## 为什么之前能用？

**假设**：
- 右手键盘可能之前运行的是**普通版固件**
- 普通版固件可能通过其他方式（直接USB/蓝牙连接电脑）
- 现在接收器版本要求右手键盘作为外设连接接收器
- 需要刷专用的`sofle_dongle_right`固件

## 故障排查

### 问题1：小红点完全无反应
- **检查**：是否刷了右手键盘固件？
- **解决**：刷入`right_trackpoint_dongle-u2f.bin`

### 问题2：小红点能动，但J键功能不正常
- **检查**：position码是否正确（J键 = position 36）
- **检查**：是否在正确的层（鼠标层）
- **解决**：确认最新固件（提交 `2e7271a`）

### 问题3：层切换不正常
- **检查**：自动层切换配置
- **检查**：trackpoint_listener是否启用
- **解决**：检查`sofle_dongle.dts`配置
